Matching and Factor

Definitions

  • perfect
    • If M saturate every vertices of G
  • M-alternative
    • if the edges in the path alternate between edges that are part of M and edges that are not part of M.
  • M-augmenting
    • if it is an M-alternating path and both endpoints of the path are unsaturated by M, implying both the starting and ending edges of the path are not part of M.
    • Let A and B be two sets. Then the symmetric difference A△B is all those elements in exactly one of A and B; that is, A△B=(A−B)∪(B−A).
  • Vertex Cover
    • A vertex cover Q for a graph G is a subset of vertices so that every edge of G has at least one endpoint in Q.
  • o(G)
    • The number of odd component in G
  • SDR
    • The abbreviation of a “system of distinct representatives”. Given a collection of finite nonempty sets , “representatives” means a collection so that is a member of set and for all
  • stable matching
    • In perfect matching, if x and y are not matched but both rank the other higher than their current partner, it’s called “unstable matching”, and “stable” matching is “stable” if there are no “unstable matching”
  • k-factor
    • Let G be a graph with spanning graph H, if H is k-regular, then H is a k-factor of G

  • Berge’s
    • A matching M of a graph G is maximum if and only if G does not contain any M-augmenting paths.
  • König-Egerváry
    • For a bipartite graph G, the size of a maximum matching of G equals the size of a minimum vertex cover for G.
  • Tutte’s
    • A graph G=(V, E) has a perfect matching if and only if for every proper subset of vertices S the number of odd components of G−S is at most |S|, that is o(G−S)≤|S|.

Gale-Shapley Algorithm

  1. Each man proposes to the highest ranking woman on his list
  2. If every woman recieve only one proposal, then output
  3. If a woman recieve more than one proposal,
    a. accepts if it is from the man she prefers above all other currently available men and rejects the rest
    b. delays with a maybe to the highest ranked and reject rest
  4. Each man propose to the highest ranked woman on his list who has not rejected him
  5. repeat 2-4 until all people have been paired

Output: Stable Matching
