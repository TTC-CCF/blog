Floating Point binary 的運算方法

表示方法 (IEEE Floating-Point Format)


  • single (32bits)

    • Exponent: 8 bits
    • Fraction (尾數): 23 bits
    • Bias: 127 bits
    • S: 正數或負數

    特性

    1. Exponent 00000000 和 11111111為保留數:
      • 00000000: 代表正負0或denormalized number
      • 11111111: 代表正負無限或NaN
    2. 最小/最大值:
      • 最小:
        • Exponent = 00000001
        • Fraction = 00000000
      • 最大:
        • Exponent = 11111110
        • Fraction = 11111111
    3. Precision:
      只跟Fraction有關 (23 bits)
      decimal digits of precision

  • double (64bits)

    • Exponent: 11 bits
    • Fraction (尾數): 52 bits
    • Bias: 1203 bits
    • S: 正數或負數

    特性

    1. Exponent 000…0 和 111…1為保留數:
      • 000…0: 代表正負0或denormalized number
      • 111…1: 代表正負無限或NaN
    2. 最小/最大值:
      • 最小:
        • Exponent = 000…01
        • Fraction = 000…00
      • 最大:
        • Exponent = 111…10
        • Fraction = 111…11
    3. Precision:
      只跟Fraction有關 (52 bits)
      decimal digits of precision
