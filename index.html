Line Messaging Api 用python寫出第一個聊天機器人

7月 26日 2023

什麼是Line Messaging Api？

越來越多商家使用Line來當作他們的官方帳號，使用大家都在用的Line來行銷，可以是推播最近的優惠，或是得到客戶的資料。有了Line Messaging Api，大家都可以建造屬於自己的聊天機器人，裡面提供超多功能可以玩，而且完全免費！
支援的功能有：

  1. 自訂回復訊息邏輯
  2. 加入圖文表單
  3. 自行劃分選單區域
  4. 自訂模板訊息
  5. 快速回覆訊息
    這裡只是列出一些常用功能，其實它能做的事還有更多，其他的功能請參考官方文檔

2023辦護照熱潮

7月 23日 2023

為什麼大家都在辦護照？

隨著疫情已經逐漸退去，經歷了近三年的疫情生活，台灣在2023年的暑假掀起了出國旅遊的熱潮，人們都想在這個暑假把之前沒有玩到的玩回來。也因為太久沒出國，護照都已經過期了，所以才會有這麼多人擠在現在辦護照。

辦護照要準備什麼？

先附上申請護照資訊連結，護照分成兩種辦法：

  • 現場辦
    直接拿著你的資料到現場辦理
  • 網路預約
    先在網路預約系統約個日期與時段，再拿你的資料到現場的網路預約處辦理
    網路預約申辦系統連結

需要準備的資料有：

  1. 簡式護照資料表 下載連結
  2. 舊護照 (如果你的舊護照還沒過期的話)
  3. 身分證或居留證
  4. 符合護照格式的大頭照

此處僅列出18歲以上所需要帶的資料，其它詳細規定請看申請換發普通護照說明書

網路預約辦理日細節

因為我本人是網路預約的，2023六月底的時候去系統上面看，最近的只能預約到七月中的時段，非常搶手，建議大家早點預約辦理。
到了預約日當天，我就直接到台北的外交部領事事務局辦理，一開始還走錯棟，視力比較不好一點，這裡先附上地圖，希望大家不要跟我一樣走錯棟。

HTB系列：Linux Module

7月 22日 2023

Linux Fundamental

對於Linux一竅不通的我很想找機會來好好學習它，現在放假了，當然要來好好學一下，透過上次介紹的HTB Academic平台來學習。

Linux Fundamental這個Module有配給你一台一天可以用兩個小時的雲端虛擬機，它可以直接ssh進練習的目標機器，不需要再設定VPN，但是呢!它整個課程只能使用兩小時真的是不太夠，我都還沒到課程的一半它就結束了，只好甘願用自己的虛擬機來連線。

教學方式

這個課程有分成多個章節，可以讓你循序漸進的熟悉linux指令，我自己的linux基礎並不是很深，所以對才剛接觸linux的我來說，這個課程鰻花時間的。每一章都有一些題目，有分兩種，實作和簡答。

  • 實作題：
    要你連進目標機器拿到題目要的資訊
  • 簡答題：
    單純回答題目而已，通常google就會有答案了

Vpn連線

補充一下，我是用自己電腦上面的虛擬機來練進目標機器的vpn的，使用的作業系統是Debian，先到題目附近下載vpn檔案之後就在自己的terminal裡面打上
sudo openvpn 'vpn檔案路徑'
然後再點’spawn the target’得到目標機器的ip位置，就可以執行
ssh htb-student@ip.address
記得虛擬機的網路設定要設定成橋接模式vpn才連得通喔!

HTB系列介紹

7月 13日 2023

什麼是HTB?

HTB Hack The Box，是一個可以讓你學習各種駭客技巧，讓你變成一個資安大佬的地方。有分成四種地方

  • HTB Academic
  • HTB Labs
  • HTB Business
  • HTB CTF

這些功能就是顧名思義而已，有針對學生的Academic和針對企業員工訓練的Business版本，還有像是CTF的競賽區，Labs就是可以進行各種操作的地方。HTB就是給你一個目標，你要去達成它，這些目標都是對一台虛擬機作一些事情或是得到機器裡面的資料，其實就是CTF的形式，因為最近學習慾爆棚，一直對CTF很嚮往的我終於要來試試看這個酷東西了。
我最近主要是在Academic還有Labs的地方活動，這些網站內以類似遊戲的感覺來鋪陳，讓你覺得好像每完成一個任務，就離破關更進一步，是蠻讓人想一直解下去的，解完也會有很大的成就感。

Academic

裡面有數不清的課程給你選(它的課程叫做module)，每一個課程都用看起來很酷的封面包裝，讓你真的很想買下去! 沒錯，課程是需要購買的!，用站內的一種貨幣，一開始他會給你70顆綠方塊(? ，然後這些綠方塊可以透過課程中的實作部分免費得到，但是我看到現在好像也沒有課程是可以賺綠方塊的，頂多就是你花多少綠方塊拿回多少這樣，好像真的要花錢才能買到更進階的課程。

除了自己選課程，你還可以選擇他們預先幫你挑好的課程包(叫做Path)，每個課程包都會是有個主題，有點你上完這些課，你就可以精通這個技術的感覺，學起來感覺也會蠻有方向的。

目前我有買了個學Linux基礎的課程，希望可以在裡面獲得許多札實的硬知識! (升大三了再不熟Linux就完蛋)!

之後應該會持續紀錄各種在HTB裡學到的新東西，期待之後的文章!
